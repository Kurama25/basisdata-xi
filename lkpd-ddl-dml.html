Model Pembelajaran

PROJECT BASED LEARNING (PjBL)

Project Based Learning adalah model pembelajaran yang menggunakan proyek/kegiatan sebagai media pembelajaran untuk mencapai kompetensi. Peserta didik melakukan eksplorasi, penilaian, interpretasi, sintesis, dan informasi untuk menghasilkan berbagai bentuk hasil belajar melalui kerja kelompok.

üéØ Kompetensi Dasar

Peserta didik mampu memahami dan menerapkan perintah SQL untuk membuat, mengubah, dan mengelola database serta tabel (DDL), serta mampu memanipulasi data dengan operasi INSERT, SELECT, UPDATE, dan DELETE (DML).

‚úÖ Tujuan Pembelajaran

  1. Peserta didik dapat membuat database dan tabel menggunakan perintah DDL
  2. Peserta didik dapat mengubah struktur tabel dengan perintah ALTER TABLE
  3. Peserta didik dapat menambahkan data ke dalam tabel menggunakan INSERT
  4. Peserta didik dapat menampilkan data dengan berbagai kondisi menggunakan SELECT
  5. Peserta didik dapat mengubah dan menghapus data dengan UPDATE dan DELETE

üìã Petunjuk Pengerjaan

  1. Baca setiap pertanyaan dengan teliti
  2. Kerjakan soal secara berurutan
  3. Tuliskan query SQL Anda pada kolom jawaban yang tersedia
  4. Uji setiap query di MySQL/phpMyAdmin untuk memastikan hasilnya benar
  5. Jawaban dapat diketik langsung pada form atau ditulis di buku
